# SCM Example (Github and Build Systems)

## Ant build system

Build file: `build.xml`

To compile the project:
```
ant build
```
Compilation output dir: `target`


To run the tests:
```
ant test
```


## Maven build system


Build file: `pom.xml`

To compile the project:
```
mvn compile
```
Compilation output dir: `target`


To run the tests:
```
mvn test
```


## Gradle build system


Build file: `gradle.build`

To compile the project:
```
gradle classes
```
Compilation output dir: `build`


To run the tests:
```
gradle test
```
